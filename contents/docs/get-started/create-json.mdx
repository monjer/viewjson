# Creating and Parsing JSON: Writing and Extracting Data

In the first two guides, we introduced the basic concepts of JSON and its syntax structure. Now, we will learn how to create and parse JSON data in programming, which is a core skill for handling JSON data.

## Creating JSON Data

Creating JSON data usually means converting data structures from a program into a JSON-formatted string. This process is called serialization. Most modern programming languages provide libraries to simplify this process.

### Creating JSON in JavaScript

JavaScript objects can be directly converted into JSON strings using the `JSON.stringify()` method.

**Example:**

```javascript
// Create a JavaScript object
var user = {
  name: "Kimi",
  age: 30,
  isAdmin: false
};

// Convert the object to a JSON string
var jsonString = JSON.stringify(user);
console.log(jsonString);
// Output: {"name":"Kimi","age":30,"isAdmin":false}
```

### Creating JSON in Other Languages

In other programming languages, such as Python, Java, or C#, there are similar libraries to create JSON strings.

**Python Example:**

```python
import json

# Create a Python dictionary
user = {
  "name": "Kimi",
  "age": 30,
  "isAdmin": False
}

# Convert the dictionary to a JSON string
json_string = json.dumps(user)
print(json_string)
# Output: {"name": "Kimi", "age": 30, "isAdmin": false}
```

## Parsing JSON Data

Parsing JSON data means converting a JSON-formatted string back into a data structure that the program can handle. This process is called deserialization.

### Parsing JSON in JavaScript

In JavaScript, you can use the `JSON.parse()` method to convert a JSON string into a JavaScript object.

**Example:**

```javascript
// Assume we have a JSON string
var jsonString = '{"name":"Kimi","age":30,"isAdmin":false}';

// Convert the JSON string to a JavaScript object
var user = JSON.parse(jsonString);
console.log(user);
// Output: {name: "Kimi", age: 30, isAdmin: false}
```

### Parsing JSON in Other Languages

In other programming languages, there are corresponding libraries to parse JSON strings.

**Python Example:**

```python
import json

# Assume we have a JSON string
json_string = '{"name": "Kimi", "age": 30, "isAdmin": false}'

# Convert the JSON string to a Python dictionary
user = json.loads(json_string)
print(user)
# Output: {'name': 'Kimi', 'age': 30, 'isAdmin': False}
```
